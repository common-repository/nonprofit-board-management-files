== Description ==

**IMPORTANT!**
This plugin extends the functionality of the "Nonprofit Board Management Plugin" by Wired Impact


The purpose of this plugin is to extend the functionality of the Nonprofit Board Management Plugin by Wired Impact to 
include Agenda and Minutes PDFs



**Event Changes**

* Adds Minutes PDF Upload to Event
* Adds Agenda PDF Upload to Event
* Adds Location Label to Event

**Shortcode**

`
[meetings]
`
